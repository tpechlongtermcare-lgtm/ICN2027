Four logic types, one auditable engine
Every criterion from the national infection-control manual is encoded as one of four explicit logic types. Nothing is a black box — each verdict can be traced back to the rule that produced it.
60×20Care-measure matrix
Sixty care actions are cross-tagged against twenty clinical contexts (device type, organism, age, cluster signal…). When a case is confirmed, the engine ranks and returns the most relevant, context-specific infection-control measures — drawn from the same manual — so the nursing response is timely and consistent.
